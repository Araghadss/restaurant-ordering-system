# 🍽️ YUM YUM – Restaurant Ordering Website

## 🔍 Project Overview

YUM YUM is a web-based restaurant ordering platform that allows users to browse restaurants, explore menus, add meals to their cart, and evaluate restaurants through ratings and feedback.

The system also includes an **owner dashboard** where restaurant owners can manage meals and offers. The goal of the platform is to provide a simple and interactive interface that improves the online food ordering experience.

---

## 🎯 Features

### Customer Features

* Browse restaurants and menus
* View meals by categories (chicken, salad, desserts)
* Sort meals by name or price
* Add meals to cart
* Adjust quantities of meals
* Checkout and submit restaurant evaluations

### Owner Features

* Owner dashboard to manage restaurant content
* Add new meals with name, description, calories, price, and image
* Manage restaurant offers
* Delete or update offers

### System Features

* Cart system for selected meals
* Restaurant rating and feedback system
* Responsive web layout
* Light and dark theme support
